WeIdentity JAVA SDK
===================

--------------

总体介绍
--------

WeIdentity Java SDK提供了一整套对WeIdentity进行管理操作的Java库。
目前，SDK支持本地密钥管理、数字身份标识（WeIdentity DID）管理、
标准化凭证（WeIdentity Credential）管理、 授权机构（Authority
Issuer）管理、 标准化模板（CPT）管理等功能，
同时也提供基于FISCO-BCOS的区块链交互、智能合约的部署与调用。
未来还将支持更丰富的功能和应用。
